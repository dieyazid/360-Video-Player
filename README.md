## Run Locally

Clone the project

```bash
  git clone git@github.com:mohammedyazid/360-Video-Player.git
```

Go to the project directory

```bash
  cd 360-Video-Player/myapp
```

Install dependencies

```bash
  flutter pub get
```

Start the app

```bash
  flutter run
```


## Running Tests

To run tests, run the following command

```bash
  cd myapp/test
```
```bash
  flutter test "testfilename.dart"
```
